# 生成-回溯理论（Generation-Retrospective-Theory）
本仓库收录了“生成-回溯理论（Generation-Retrospective-Theory）”从最初想法到理论雏形的完整发展过程。
该理论旨在描述时间内部行动者在面对未来时的选择体验、决策逻辑与实践必然性，提出“未来处于回溯唯一但当下开放状态”的核心观点： 
- 历史轨迹在生成完成后形成唯一记录；
- 当下行动者的选择过程真实存在，并对未来生成具有因果影响；
- 知晓或预测行为可作为生成的新变量，动态影响实际结果而非仅是心理反馈。

注：本文为个人思考与概念模型，仅供哲学探索与讨论参考。非正式论文，未经过系统学术验证。
